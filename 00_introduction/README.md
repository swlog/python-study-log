# 00. 프로그래밍 기초 개념

## 1. 프로그램이란?
컴퓨터가 입력을 받아 처리하고, 결과를 출력하기 위해 차례대로 작성된 명령어의 집합

**일상 속 예시**
- 요리 레시피
- 악보
- 연극 대본

## 2. 프로그래밍이란?
문제를 논리적으로 분해하고, 컴퓨터가 이해할 수 있는 언어로 해결 절차를 표현하는 과정

## 3. 코드 및 코딩
- **코드**: 프로그래밍 언어로 작성된 명령어 텍스트
- **코딩**: 코드를 작성하는 행위

## 4. 프로그래밍 언어란?
사람과 컴퓨터가 소통하기 위한 언어

**종류**
- Python, JavaScript, Java, C++ 등

## 5. 파이썬을 배우는 이유
- 문법이 간결하여 개념과 로직에 집중할 수 있음
- OPP, 자료구조 등 핵심 개념을 구현하며 학습하기에 적합
- 풍부한 라이브러리와 커뮤니티를 통해 학습 자료 접근성이 높음
- 알고리즘, 자동화, 데이터 처리 등 다양한 분야의 기반 언어로 활용됨

## 6. 데이터(자료)란?
컴퓨터가 처리하는 모든 정보

**예시**
- 숫자: 100, 3.14
- 문자: "안녕하세요"
- 참/거짓: True, False
